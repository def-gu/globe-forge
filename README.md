# Globe Forge

Модуль для [Foundry VTT](https://foundryvtt.com/) v13: интерактивные глобусы игровых миров. Сцена-глобус с векторными тайлами и свободным зумом от планеты до региона; глобусы описываются декларативными манифестами и создаются пользователем. Первый встроенный мир — Голарион на данных проекта [pf-wikis/mapping](https://github.com/pf-wikis/mapping).

## Планируемые возможности

- Токены, линейка с правилами путешествий, заметки и переход на страницы вики.
- Контуры регионов и стран; двойной клик — привязанная заметка или переход в обычную сцену региона/города.
- Туман войны стратегического масштаба с режимом «выцветшая бумажная карта».
- Несколько источников данных с автоматическим переключением; полная работа офлайн с локальным файлом.
- Создание собственных глобусов, включая нарезку плоских карт.
